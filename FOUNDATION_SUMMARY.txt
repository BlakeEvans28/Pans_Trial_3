✅ Appeasing Pan Phase:
- Simultaneous card play
- Trump order (Walls > Traps > Ballista > Weapons)
- Tie-breaking (highest value wins)
- Request selection (winner first)

✅ Card Effects:
- Wall: Blocks movement
- Trap: Take damage, steal highest opponent card
- Ballista: Move anywhere in row/column without effect
- Weapon: Required for combat

✅ Combat:
- Automatic on same-cell collision
- Both players play highest card if they have weapon
- Damage dealt from hand to damage pile

✅ Damage Tracking:
- Card values: Ace=1, 2-10=face, Queen=11, King=12
- Accumulating damage pile
- Win condition at 25+ damage

DESIGN FOR AI INTEGRATION
==========================

The engine exposes the interface needed for AI:

1. get_legal_moves(player_id) -> list[str]
   - All valid actions from current position
   
2. apply_action(action: Action) -> bool
   - Execute action and update state
   
3. check_game_over() -> bool
   - Terminal state detection
   
4. get_player_hand(player_id) -> list[Card]
5. get_damage_total(player_id) -> int
   - State queries for decision making

This design allows future implementation of:
- HumanAgent (current)
- HeuristicAgent
- RLAgent (with gymnasium)
- Self-play simulations
